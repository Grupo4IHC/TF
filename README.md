# AquaAlert

## Descripción del Proyecto
**AquaAlert** es una aplicación web diseñada para empoderar a los ciudadanos mediante información en tiempo real sobre la calidad del agua. La plataforma permite monitorear cisternas y redes de agua, recibir alertas ante posibles riesgos sanitarios y fomentar un consumo responsable y seguro. AquaAlert integra funcionalidades de comunicación entre usuarios y autoridades, promoviendo la transparencia y la participación ciudadana en la vigilancia ambiental.

## Objetivos del Proyecto
- Proporcionar información precisa y actualizada sobre la calidad del agua.
- Alertar a la población sobre posibles riesgos sanitarios en tiempo real.
- Facilitar la comunicación entre usuarios y autoridades para una rápida gestión de incidencias.
- Promover la educación y conciencia sobre el consumo seguro de agua.
- Fomentar la participación ciudadana en la vigilancia ambiental.

## Segmentos Objetivo
- **Ciudadanos** que desean información confiable sobre la calidad del agua en sus hogares y comunidades.  
- **Autoridades locales y municipales** encargadas de gestionar la calidad del agua y responder a emergencias.  
- **Instituciones educativas y de salud** interesadas en promover la educación sobre el consumo seguro del agua.  
- **Organizaciones ambientales y sociales** que buscan datos en tiempo real para análisis y toma de decisiones.

## Principales Características
- **Monitoreo en tiempo real**: Visualización de la calidad del agua en diferentes zonas mediante mapas interactivos.  
- **Alertas y notificaciones**: Sistema de alertas automáticas para informar sobre posibles contaminaciones o interrupciones del suministro.  
- **Formulario de contacto**: Canal directo para que los usuarios reporten incidencias o realicen consultas.  
- **Gestión de perfil de usuario**: Registro e inicio de sesión con personalización de experiencia.  
- **Panel de administración**: Permite a las autoridades revisar reportes de usuarios y tomar decisiones informadas.  
- **Diseño accesible y responsivo**: Adaptable a diferentes dispositivos y fácil de usar.

- **Frontend**: HTML5, CSS3, JavaScript, FontAwesome, Google Fonts.

## Uso
1. Abrir el archivo `index.html` en un navegador web moderno.  
2. Navegar por las secciones de **Inicio**, **Sobre Nosotros**, **Contacto** 
3. Registrar un usuario o iniciar sesión para acceder a funcionalidades personalizadas.  
4. Usar el formulario de contacto para enviar consultas o reportes.
